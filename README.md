# SideKick Django

## Pokretanje lokalno

Instaliraj zavisnosti:

```bash
pip install -r requirements.txt
```

Pokreni Django server:

```bash
python manage.py runserver
```

Ako koristiš Windows Python launcher:

```bash
py manage.py runserver
```

Otvori aplikaciju u browseru:

```text
http://127.0.0.1:8000/
```
